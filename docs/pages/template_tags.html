{mojo:blog:entries}

The entries tag is the most complex tag in MojoBlog, and allows you to display, loop through and output your blog entries. It also contains functionality for pagination and category listing.


Generic Parameters

limit="x"
Only display x entries
orderby="date"
Order by id, date, title and content
sort="asc"
Sort ascendingly or descendingly
page="about|index"
Only output this tag only on a page, or a bar separated list of pages. Useful if you're not using a separate layout for your blog.
global="yes"
Show this tag globally
date_format="Y-m-d H:i:s"
The formatting for the entry date tag. Uses PHP's date formatting.
no_posts="There are no posts!"
Text/HTML to display when no posts exist
status="not draft"
Show posts from a specific status, either published, draft or review. Defaults to published

Entry/Category Parameters

entry_id="2"
A specific entry ID to show
entry_id_segment="3"
The number of the URL segment that holds the entry ID
entry_url_title_segment="3"
The number of the URL segment that holds the entry URL title
no_posts_404="yes"
Throw a 404 if there are no posts
category="not mojoblog|expressionengine"
Show posts from a specific category or multiple categories.
category_segment="3"
The number of the URL segment that contains the category URL name
category="cat_url_name"
The URL name of a category you wish to show
excerpt_words="20"
The number of words to chop off at in the excerpt

Pagination Parameters

paginate="yes"
Enable pagination
per_page="5"
How many entries to show per page
pagination_trigger="p"
The trigger in the URL to define the page number
paginate_once="yes"
Only ever show the pagination once

While there are lot of parameters to choose from, this gives you fine-grained control over the output of the tag. In addition to these parameters, this tag supports the following variables:


Inside the {entries}{/entries} tag

These variables sit inside a {entries}{/entries} loop to display the actual entry data.

{id}
The entry ID
{title}
The entry's title
{url_title}
The entry's URL title
{url_title_path}
A URL, generated from the current page and the entry's URL title
{content}
The entry content
{excerpt}
An excerpt of the entry content, trimmed off at excerpt_words
{status}
The entry status
{author}
The entry author
{category_name}
This entry's category name
{category_url_name}
This entry's category URL name
{category_id}
This entry's category ID
{if category}{/if} or {if not category}{/if}
Display content if post has a category (or not)
{if single_entry_page}{/if} or {if not single_entry_page}{/if}
Display content if this is a single entry page (or not)

Category Variables

These variables are for category listing pages.

{category_name}
The current category name
{category_url_name}
The current category URL name
{category_id}
The current category ID
{if category_page}{/if} or {if not category_page}{/if}
Is this a category listing page?

Pagination

Pagination links are shown outside of the {entries} tag in a separate {pagination} tag. The {pagination} tag has the following variables:

{first_page_url}
The URL to the first page
{if prev_page}
Is there a previous page?
{prev_page_url}
The URL to the previous page, if there is one
{current_page}
The current page number
{total_pages}
The total number of pages
{if next_page}
Is there a next page?
{next_page_url}
The URL to the next page
{last_page_url}
The URL to the last page

Basic Example

The following example is for a basic blog that shows the last ten entries.

{mojo:blog:entries limit="10" orderby="date"}
  <h1>Blog</h1>

  {entries}
          <h2>{title}</h2>
          <p>Author: {author}</p>

          {content}
  {/entries}
{/mojo:blog:entries}

+ Pagination

The next example builds upon the basic example, but adds pagination (new code in bold).

{mojo:blog:entries limit="10" orderby="date" paginate="yes"}
  <h1>Blog</h1>

  {entries}
      <h2>{title}</h2>
      <p>Author: {author}</p>

      {content}
  {/entries}

  {pagination}
      <p>
        <a href="{first_page_url}">First Page</a>
        {if prev_page} - <a href="{prev_page_url}">Previous Page</a>{/if} - 
        Page {current_page} of {total_pages} - 
        {if next_page}<a href="{next_page_url}">Next Page</a> - {/if}
        <a href="{last_page_url}">Last Page</a></p>
  {/pagination}
{/mojo:blog:entries}

+ Single Entry Page

We now add support for single entry pages, all within the one tag

{mojo:blog:entries limit="10" orderby="date" paginate="yes" entry_url_title_segment="3"}
  <h1>Blog</h1>
  
  {entries}
      {if single_entry_page}
        <h2>{title}</h2>
      {/if}
      
      {if not single_entry_page}
        <h2><a href="{url_title_path}">{title}</a></h2>
      {/if}
      
      <p>Author: {author}</p>

      {content}
  {/entries}

  {pagination}
      <p>
        <a href="{first_page_url}">First Page</a>
        {if prev_page} - <a href="{prev_page_url}">Previous Page</a>{/if} - 
        Page {current_page} of {total_pages} - 
        {if next_page}<a href="{next_page_url}">Next Page</a> - {/if}
        <a href="{last_page_url}">Last Page</a></p>
  {/pagination}
{/mojo:blog:entries}

+ Categories

We'll now add category support!

{mojo:blog:entries limit="10" orderby="date" paginate="yes" entry_url_title_segment="3" category_segment="3"}
  <h1>Blog {if category_page}entries in {category_name}{/if}</h1>

  {entries}
      {if single_entry_page}
        <h2>{title}</h2>
      {/if}

      {if not single_entry_page}
        <h2><a href="{url_title_path}">{title}</a></h2>
      {/if}

      <p>Author: {author}</p>
      <p>Category: {if category}<a href="{mojo:site:link page="blog/category/{category_url_name}"}">{category_name}</a>{/if}{if not category}No Category{/if}</p>

      {content}
  {/entries}

  {pagination}
      <p>
        <a href="{first_page_url}">First Page</a>
        {if prev_page} - <a href="{prev_page_url}">Previous Page</a>{/if} - 
        Page {current_page} of {total_pages} - 
        {if next_page}<a href="{next_page_url}">Next Page</a> - {/if}
        <a href="{last_page_url}">Last Page</a></p>
  {/pagination}
{/mojo:blog:entries}

{mojo:blog:categories}

The categories tag allows you to loop through your blog's available catagories. It takes three optional parameters, limit="", orderby="" and sort="", all of which are self explanatory. There are four variables available, {id}, {name}, {url_name} and {entries}. Here's an example:

<h2>Categories</h2>
  
<ul>
  {mojo:blog:categories}
    <li><a href="{mojo:site:link page="blog/category/{url_name}"}">{name}</a> has {entries} entries.</li>
  {/mojo:blog:categories}
</ul>

{mojo:blog:disqus}

The disqus tag makes it easy to integrate the Disqus commenting system. It takes one required parameter, shortname="", your Disqus site shortname, and one optional parameter, entry_id="" which pulls in the comments for a specific entry (for use inside of {mojo:blog:entries}).

<h1>Comments for post #1</h1>
{mojo:blog:disqus shortname="sparkplugs" entry_id="1"}

{mojo:blog:rss_url}

Outputs the URL to your RSS feed. Takes two parameters, both optional, limit="" and link_page="".

{mojo:blog:rss_url limit="15" link_page="about"}
